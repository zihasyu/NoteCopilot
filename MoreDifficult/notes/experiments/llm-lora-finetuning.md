# LLM微调实验记录 - LoRA方法

## 实验目标

使用LoRA（Low-Rank Adaptation）方法对Llama-2-7b模型进行微调，提升其在中文问答任务上的表现，同时降低显存占用。

## 实验日期

2024-03-15 ~ 2024-03-20

## 实验环境

- **GPU**: NVIDIA A100 40GB
- **框架**: PyTorch 2.0 + HuggingFace Transformers
- **基础模型**: meta-llama/Llama-2-7b-hf
- **数据集**: 中文问答数据集（50k样本）

## 实验参数

| 参数 | 值 | 说明 |
|------|-----|------|
| LoRA rank | 16 | 低秩矩阵的秩 |
| LoRA alpha | 32 | 缩放参数 |
| LoRA dropout | 0.05 | Dropout率 |
| Learning rate | 2e-4 | 学习率 |
| Batch size | 4 | 批次大小 |
| Epochs | 3 | 训练轮数 |
| Max length | 512 | 最大序列长度 |

## 实验结果

### 训练过程

```
Epoch 1/3: Loss = 2.34 -> 1.56
Epoch 2/3: Loss = 1.56 -> 1.12
Epoch 3/3: Loss = 1.12 -> 0.89
```

### 性能对比

| 指标 | 微调前 | 微调后 | 提升 |
|------|--------|--------|------|
| 准确率 | 62.3% | 74.5% | +12.2% |
| BLEU-4 | 18.5 | 24.3 | +5.8 |
| ROUGE-L | 35.2 | 42.1 | +6.9 |

### 资源占用

| 配置 | 显存占用 | 训练时间 |
|------|----------|----------|
| Full fine-tuning | 28GB | 4.5h |
| LoRA (r=16) | 14GB | 2.1h |
| **节省** | **50%** | **53%** |

## 关键发现

1. **LoRA有效性**: 仅训练0.16%的参数，实现与全量微调相近的效果
2. **显存优化**: 显存占用减少50%，可在消费级GPU上训练
3. **训练速度**: 训练时间缩短一半，迭代效率提升

## 经验教训

- Rank选择：r=16在效果和效率间达到较好平衡
- Alpha设置：alpha=2*r是经验最优值
- 数据质量：数据清洗对最终效果影响显著
